Hello!
this is an information extraction app based on PaddleNLP.

信息拆分
/get 返回拆分结果 eg.
data body: { 'text': ['{{egtext}}'] }
返回结果 {{exmp1}}
/gc 按默认格式拆分后，返回整理后的结果 parameters
getbymTrue(default) - 返回出生年月:str
False - 返回出生年:int
dealblendFalse(default) - 是否分段分解
会自动识别学习经历、工作经历，并分别进行信息提取，提高信息抽取率（略微增加运行时间）
priorlayerbool/int:False(default) - 是否优先采用分段分解的结果
2 - 仅返回分段分解的结果
chopsentencebool/str: False(default) - 是否分句分解
- 或 用于分句的标点(例如: "；")
puncbystr/list: "；。，"(default) - 分段的节点
返回结果 字典：
1.（固定键）性别、籍贯、出生年(getbym为True则 出生年月)

2.（非固定键）单位、职务、行业、地址、学习经历(值为列表，列表元素固定键: 学校、院系、专业、入学年份、毕业年份、学历)、任职学校、毕业学校、曾任学校(曾任职的学校)
/gs 可按需调用预设置好的schema进行信息抽取 parameters
schema 自定义schema(list/dict) 或 "工作经历" / "学习经历" / "基本信息"
default: 籍贯、性别、年龄、 行业、职务、单位、地址、 毕业院校、专业、入学年份、毕业年份、学位、学历
getbym同上
returnmodelopTrue - 返回模型原始输出
False(default) - 返回输入schema对应的字典（多个值用“、”连接）
舆情分析
/qgqx 情感倾向 default schema: "情感倾向[正向，中性，负向]" 分类任务，schema类型应为String，参照左边格式，不同类型用"，"分开
/pjcq 评价抽取 eg. body: { 'text': ['热烈祝贺06级物理系李媛媛校友荣获“第十四届全国中学物理青年教师教学大赛”高中组一等奖：06级李媛媛校友以重庆市高中物理青年教师优质课大赛一等奖第一名的优异成绩，代表重庆市参加全国大赛，荣获全国中学物理教学大赛高中组一等奖。'] }
返回结果： {'成绩': {'情感倾向': {'probability': 0.9671025016000385, 'text': '正向'}, '观点词': '优异'}}
